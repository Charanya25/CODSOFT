## üß† Projects Included

### 1. Titanic - Survival Prediction
- Dataset: [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic)
- Goal: Predict which passengers survived based on their features.
- Techniques: Logistic Regression, Random Forest, XGBoost
- Evaluation Metrics: Accuracy, Precision, Recall, F1-score

### 2. Iris Flower Classification
- Dataset: [Iris Dataset](https://www.kaggle.com/datasets/uciml/iris)
- Goal: Classify flowers into three species based on sepal and petal measurements.
- Techniques: Decision Tree, KNN, SVM
- Evaluation Metrics: Accuracy, Confusion Matrix

### 3. Credit Card Fraud Detection
- Dataset: [Credit Card Fraud Detection Dataset](https://www.kaggle.com/mlg-ulb/creditcardfraud)
- Goal: Detect fraudulent transactions using imbalanced data handling.
- Techniques: SMOTE, Logistic Regression, Random Forest
- Evaluation Metrics: Precision, Recall, ROC-AUC
